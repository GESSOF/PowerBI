Resumen Ejecutivo

¿Qué es Power BI?

Microsoft Power BI es una plataforma de Business Intelligence que permite conectar y transformar datos de múltiples fuentes, crear visualizaciones interactivas y dashboards profesionales, compartir insights de forma segura y tomar decisiones basadas en datos en tiempo real.

Power Query

Motor ETL integrado que conecta, limpia y transforma datos de diversas fuentes. Utiliza el lenguaje M para transformaciones avanzadas, automatiza la preparación de datos y garantiza consistencia en el análisis.

DAX (Data Analysis Expressions)

Lenguaje de fórmulas para crear medidas y columnas calculadas. Permite realizar cálculos complejos, funciones de tiempo, agregaciones y filtros contextuales, potenciando el análisis con KPIs personalizados.
